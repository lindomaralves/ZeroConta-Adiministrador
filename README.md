# ZeroConta (modo de teste)

Este é um formulário de cadastro básico que envia os dados para o cenário Make usando o módulo:

- `Webhook`
  - **Custom Webhook**

## Objetivo

Testar a integração entre o frontend (formulário) e o cenário do Make antes de usar dados reais.

## Fluxo inicial

1. Usuário preenche os dados no formulário.
2. Os dados são enviados via JSON para o Custom Webhook no Make.
3. O Make processa e em breve envia para o Google Sheets (em próxima etapa).
