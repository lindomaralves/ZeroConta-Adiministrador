# ZeroConta - MicroSaaS de Cadastro

Este projeto é uma aplicação de cadastro simples conectada ao Make (Integromat) e Google Planilhas. Os dados preenchidos são enviados automaticamente via Webhook para o cenário no Make.

## ✅ Arquivos do Projeto

1. **index.html** - Estrutura da interface de cadastro.
2. **style.css** - Estilização da interface.
3. **script.js** - Envio de dados via Webhook.
4. **README.md** - Este arquivo explicativo.

## 🧠 Campos do formulário e da planilha:
- Nome Completo
- Tipo Usuário
- CPF/CNPJ
- Email
- WhatsApp
- Endereço
